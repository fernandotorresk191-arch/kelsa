# 🔧 Backoffice Kelsa - Руководство администратора

## Что такое Backoffice?

Backoffice (бэкофис) — это скрытая от обычных пользователей часть приложения, предназначенная для менеджеров и администраторов. Здесь вы можете управлять заказами, товарами, отслеживать продажи и аналитику.

**Доступ:** `https://bo.gokelsa.ru` или `/admin`

---

## 🚀 Начало работы

### 1. Вход в систему

Перейдите на страницу логина:
```
https://bo.gokelsa.ru/login
```

Введите свои учетные данные (email и пароль, выданные администратором).

### 2. Главный экран

После входа вы увидите:
- **Панель управления** — основная статистика
- **Заказы** — управление заказами
- **Товары** — управление каталогом
- **Аналитика** — отчеты и статистика

---

## 📊 Панель управления (Dashboard)

На главной странице вы видите:

### KPI Блоки
- **Всего заказов** — количество всех заказов
- **Общая выручка** — сумма всех продаж
- **Выручка сегодня** — доход за сегодняшний день
- **Средний чек** — средняя сумма заказа

Распределение заказов по статусам:
- 🆕 **NEW** (Новый) — только что созданы
- ✅ **CONFIRMED** (Подтвержден) — подтверждены менеджером
- 📦 **ASSEMBLING** (Собирается) — находятся на сборке
- 🚚 **ON_THE_WAY** (В пути) — в доставке
- 🎁 **DELIVERED** (Доставлен) — успешно доставлены
- ❌ **CANCELED** (Отменен) — отменены

### Топ товаров
Таблица с самыми продаваемыми товарами.

## 📦 Управление заказами

### Просмотр списка заказов

Перейдите в раздел **Заказы**.

Здесь вы видите:
- Номер заказа (#)
- Имя и телефон клиента
- Статус заказа
- Сумму заказа
- Дату создания

### Фильтрация по статусу

Используйте выпадающий список для фильтрации заказов по статусам.

### Просмотр деталей заказа

Кликните на **"Просмотр →"** для открытия деталей заказа.

#### На странице заказа:

**Информация о заказе:**
- Клиент
- Телефон
- Адрес доставки
- Комментарий клиента (если был)

**Товары в заказе:**
Таблица со всеми товарами в заказе:
- Название
- Цена
- Количество
- Сумма

**История статусов:**
Все изменения статуса с датой и комментариями

**Управление статусом:**
- Выберите новый статус из выпадающего списка
- Добавьте комментарий (опционально)
- Нажмите "Обновить статус"

**Печать накладной:**
Кликните "🖨️ Печать накладной" для вывода документа на печать.

## 🛍️ Управление товарами

### Просмотр списка товаров

Перейдите в раздел **Товары**.

Таблица показывает:
- Название товара
- Цену
- Остаток на складе
- Статус (активный/отключенный)
- Действия

### Добавление нового товара

1. Кликните кнопку **"+ Добавить товар"**
2. Заполните форму:
   - **Название** — название товара
   - **Slug** — уникальный идентификатор для URL (например: "moloko-domodnoe")
   - **Описание** — описание товара
   - **Цена** — цена в рублях
   - **Остаток** — количество на складе
   - **URL изображения** — ссылка на фото товара

3. Кликните **"Добавить товар"**

### Редактирование товара

1. Кликните на товар в таблице (**"Редактировать"**)
2. Измените необходимые поля
3. Кликните **"Сохранить"**

### Управление остатками

В форме редактирования товара есть поле **"Остаток (шт.)"**:
- Введите новое количество
- Нажмите "Сохранить"

Остаток обновится и будет отображаться на сайте для клиентов.

### Отключение товара

В форме редактирования товара отключите опцию **"Товар активен"** и сохраните.

Товар не будет удален, но не будет отображаться на сайте.

### Удаление товара

Кликните **"Удалить"** в таблице товаров.

Товар будет отключен (см. выше).

## 🌐 Доступ через разные домены

Backoffice доступен по адресу:
- `https://bo.gokelsa.ru` — основной домен для бэкофиса
- `https://localhost:3000/admin` — локальная разработка
- `https://yourdomain.com/admin` — если используется другой домен

### Настройка DNS

Для настройки поддомена `bo.gokelsa.ru`:

1. В панели управления домена добавьте CNAME запись:
   ```
   bo.gokelsa.ru CNAME your-server.com
   ```

2. Или A запись, если есть статический IP:
   ```
   bo.gokelsa.ru A 1.2.3.4
   ```

3. Убедитесь, что сервер обслуживает оба домена (`gokelsa.ru` и `bo.gokelsa.ru`)

## 🐛 Устранение неполадок

### Не могу войти

1. Проверьте, что вводите правильный email и пароль
2. Попросите администратора проверить, активен ли ваш аккаунт
3. Очистите кэш браузера (Ctrl+Shift+Delete)

### Заказ не обновляется

1. Перезагрузите страницу (F5)
2. Проверьте интернет соединение
3. Попробуйте зайти заново

### Товар не появляется на сайте

1. Убедитесь, что товар отмечен как "активный"
2. Проверьте, что цена больше 0
3. Попробуйте обновить кэш сайта
