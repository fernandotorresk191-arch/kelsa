#### 2. Админ-модуль с 4 контроллерами

**admin-auth.controller.ts**
```
POST /v1/admin-auth/login      — Логин администратора
GET  /v1/admin-auth/me         — Получить профиль админа
POST /v1/admin-auth/create     — Создать нового админа (только для админов)
```

**admin-orders.controller.ts**
```
GET    /v1/admin/orders              — Список заказов (с пагинацией и фильтром)
GET    /v1/admin/orders/:id          — Детали заказа
PATCH  /v1/admin/orders/:id/status   — Изменить статус заказа
GET    /v1/admin/orders/:id/print/invoice — HTML для печати накладной
```

**admin-products.controller.ts**
```
GET    /v1/admin/products            — Список товаров
GET    /v1/admin/products/:id        — Детали товара
POST   /v1/admin/products            — Добавить товар
PUT    /v1/admin/products/:id        — Редактировать товар
DELETE /v1/admin/products/:id        — Удалить товар (мягкое удаление)
PATCH  /v1/admin/products/:id/stock  — Обновить остатки
```

**admin-analytics.controller.ts**
```
GET /v1/admin/analytics/dashboard                — Панель управления (KPI, статусы, топ товаров)
GET /v1/admin/analytics/orders-stats             — Статистика по заказам
GET /v1/admin/analytics/products-sales           — Топ товаров по продажам
GET /v1/admin/analytics/revenue-analytics        — Выручка по периодам
```

## 🚀 Как начать работу

### 1. Быстрый старт (5 минут)

```bash
# Применить миграцию БД
cd backend/api && npx prisma migrate deploy

# Создать администратора (используя bcrypt хэш)
npx prisma studio  # или SQL запрос

# Запустить бэкэнд
npm run start:dev

# В новом терминале запустить фронтэнд
cd frontend && npm run dev

# Открыть http://localhost:3000/admin/login
```

### 2. Создать первого админа

Используйте Prisma Studio или SQL:
```sql
INSERT INTO "AdminUser" VALUES (
  'admin_1',
  'admin@gokelsa.ru',
  '$2b$10$...',  -- bcrypt хэш пароля
  'admin',
  true,
  NOW(),
  NOW()
);
```
